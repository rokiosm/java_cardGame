# java_cardGame

### 게임 기능 – 1vs1 스피드 카드 게임

상대방과 실시간 1대1 카드 게임 진행

승리 시 특별 보상 획득 가능

### 채팅 기능

실시간 채팅 지원

비속어 필터링 기능 탑재

비속어 입력 시 1회 경고

경고 3회 누적 시 채팅 제한

게임 승리 시 이모티콘 사용 가능

 ### 보상 시스템

게임 승수에 따라 배지(Badge) 제공

배지는 채팅창 닉네임 왼쪽에 표시

승리하면 이모티콘 사용 + 배지 적용 가능

### 예상 결과 화면

프로젝트에서 구현 예정/완료한 주요 화면들:

첫 실행 화면

게임 룰 설명 화면

설정 화면

배지 변경 화면

게임 준비 화면

상대방 대기 화면

게임 플레이 화면

비속어 사용 시 경고 팝업

승리/패배 시 결과 화면

서버 화면 (관리자용)


### 서버 구성

사용자 간 실시간 통신을 위한 서버 제공

게임 정보, 채팅 데이터, 경고 기록 등을 관리
